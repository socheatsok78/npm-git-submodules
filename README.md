# npm-git-submodules
A NPM hook for fetching git submodules

## Install

```sh
npm install --save-dev npm-git-submodules
```

## Usage

Add the following to your `package.json`:

```json
{
  "scripts": {
    "prepare": "git-submodules"
  }
}
```
